# Meetup_event_planner

This project is a part of the Senior Web Developer Nano Degree through Udacity.

### About
Web application that allows the user to create a meet-up event.

## Installation
- Installing dependencies:
```shell
npm install
```

## Running server

```shell
gulp
```

HTML5  
CSS3
